# Interchain gov

- Propose state change
- Vote on state change
- Execute state change (or msgs)

## Adding Members

### Locally (A)
1. Create proposal 1 to add member B 
2. Execute proposal 1
    1. Now there are 2 members locally (A, B)
   2. Send proposal
